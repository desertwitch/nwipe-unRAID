### nwipe (Secure Disk Eraser) with Preclear (for UNRAID)

![GitHub Tag](https://img.shields.io/github/v/tag/desertwitch/nwipe-unRAID?label=release&color=peru)

Installable via Community Applications or Direct URL on UNRAID 6.10+:

https://raw.githubusercontent.com/desertwitch/nwipe-unRAID/main/plugin/dwnwipe.plg

For the general-purpose, forked `nwipe-pc` (which includes preclear) check out:

https://github.com/desertwitch/nwipe-pc

For preclearing on another non-Unraid system check out the Docker container:

https://github.com/desertwitch/nwipe-pc/pkgs/container/nwipe-pc
